What is FlexBox?

Flexbox (Flexible Box Layout) is a CSS3 layout module designed to more efficiently arrange, align, and distribute space among elements in a container, even when their size is unknown or dynamic.


The main idea of Flexbox is to give a container the ability to alter its items' width, height, and order to best fill the available space. A Flexbox container can stretch items to fill free space or shrink them to prevent overflow.

Columns with flex-containers
Flex Container
An element becomes a flex container when you set the display property to flex or inline-flex. All its direct children automatically become flex items.
Element Direction
The flex-direction property defines the direction of the main axis along which flex items are placed. Possible values: row, row-reverse, column, column-reverse.
Axis Alignment
The justify-content property defines the alignment of items along the main axis. Possible values: flex-start, flex-end, center, space-between, space-around, space-evenly.
Cross-axis Alignment
The align-items property defines the alignment of items along the cross axis. Possible values: stretch, flex-start, flex-end, center, baseline.
Benefits of Flexbox
Ease of Use
Easy to Learn
Flexbox provides a simple and intuitive way to create flexible layouts. Just a few properties allow you to create complex responsive designs that previously required many CSS hacks.

Responsiveness
Layout Flexibility

Flexbox items automatically adapt to the available space, making creating responsive design much easier. Elements can shrink and expand according to the container dimensions.


This is especially useful for creating layouts that should display correctly on various devices and screen sizes.
